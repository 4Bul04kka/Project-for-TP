# Веб сайт Lunaris
http://lunaris.ddns.net/

Для того чтобы зайти под Admin необходимо в футере сайта нажать на логотип Lunaris.
При нажатии на него появляется модальное окно с входом для админа.
Логин admin1 пароль password
Чтобы выйти нужно снова нажать на логотип Lunaris в футере сайта.
Появится модальное окно с кнопкой выхода.
